##아쉽게도 좌표를 직접 입력해야 합니다. 다음 버전은 이미지를 통해 실행하는 매크로로 찾아오겠습니다.##

1. 멤버확인 창에서 이름을 입력합니다.
2. 각각의 좌표를 입력합니다.(에버랜드,캐비 첫 페이지 부터 시작)
3. 창에서부터 마우스를 클릭한 채로 마우스를 이동하면 현재 좌표(x=22, y=232)가 보입니다.
4. 현재 좌표 가져오기를 누르면 아래 보이는 현재 좌표(x=22, y=232)를 쉽게 입력할 수 있습니다.
5. 매수 좌표를 입력할때는 원하는 매수 좌표가 어딘지 주의해야합니다.
6. Save 버튼을 클릭하면 좌표 풀 세트를 저장할 수 있습니다. 세이브파일은 exe파일과 같은 경로에 이름.json 으로 저장됩니다.
ex) 1매.json, 2매.json, 3매.json
7. Load 버튼을 클릭하면 원하는 세이브 파일을 로드할 수 있습니다.(단, .json은 빼고 입력합니다.)
ex) 1매
8. Play 버튼을 누르면 매크로가 실행됩니다.
9. 타이머를 이용해 원하는 시간에 매크로를 실행할 수 있습니다.
 => 실행시간(24시간 표기법)을 입력 후 타이머 실행 버튼을 누릅니다.
10. 실행 시간을 바꾸고 싶으면 실행 시간 수정 후 타이머 실행 버튼을 누르면 됩니다.
